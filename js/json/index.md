# JSON
JavaScript Object Notation
一种基于文本的数据交换格式

## JSON是什么
JSON是一种简单数据格式
键值对 —— Name/Value (Key/Value)
对象 —— Object
数组 —— Arrays

## 为什么使用JSON
如果使用过XML的话...
JSON的可读性非常好
XML那样包含很多冗余的元素标签
网络传输 速度更快 效率更高
